# core/products

Presets de produtos de credito (ver COMPLIANCE_NOTES.md secao 2): credito
pessoal, consignado, financiamento CDC/veiculo, penhor, habitacional, PJ com
fundo garantidor, desconto bancario. Cada preset pre-configura sistemas
permitidos, IOF/isencao, indexador, encargos default e tetos.

Implementacao a partir da Fase 6.
